# CareerFit AI

> 취업·공모전 데이터 기반 맞춤형 AI 포트폴리오 코치



## 프로젝트 개요



비슷한 공고들을 비교하기가 너무 귀찮다.
-> 직무별 요구 스킬을 데이터로 정리하고, 네 역량과 비교해 관련된 공고들을 제공한다. 또한 필요한 스킬이 무엇인지 알려주는 서비스도 제공할 수 있다.



## 기술 스택



| 영역 | 기술 |

|---|---|

| 백엔드 | Python, FastAPI |

| AI API | Gemini 2.5 Flash-Lite |

| 데이터 | Pandas, SQLite, ChromaDB |

| 프론트엔드 | React, Vite |

| 실행 환경 | Docker |

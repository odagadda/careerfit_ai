# PROJECT_PLAN.md — CareerFit AI 프로젝트 기획 초안



## 1. 문제 정의

비슷한 공고들을 비교하기가 너무 귀찮다.
-> 직무별 요구 스킬을 데이터로 정리하고, 네 역량과 비교해 관련된 공고들을 제공한다. 또한 필요한 스킬이 무엇인지 알려주는 서비스도 제공할 수 있다.


## 2. 사용자 정의

- 대상: 취업 준비 중인 대학교 3-4학년



- 사용 시나리오:

1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.

3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.



## 3. 데이터 계획

- 취업 공고 컬럼: 회사명, 직무명, 요구 스킬, 우대 사항, 지원 자격, 업무 내용, 마감일, 블라인드의 평판 항목, 연봉 등

- 데이터 출처: 강사 제공 목업 CSV (3일차)
